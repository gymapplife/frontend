# frontend

The frontend for our SE 464 project.

## Installation

1. ```git clone https://github.com/gymapplife/frontend.git```
2. ```cd frontend```
3. ```yarn install```

## Usage

1. ```yarn start ```
2. Visit [http://localhost:8080/](http://localhost:8080/)

## Build

```
npm install -g serve
serve -s build -p 8080
```
